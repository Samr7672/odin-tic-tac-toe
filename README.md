# Tic Tac Toe Game

A classic Tic Tac Toe game built as part of [The Odin Project](https://www.theodinproject.com/) curriculum. This web-based game allows two players to compete in the timeless strategy game right in their browser.

## Features

- **Two-Player Gameplay**: Enter custom player names and take turns playing as X and O
- **Interactive Board**: Click on the grid to make your moves
- **Win Detection**: Automatically detects wins and ties
- **Game Reset**: Restart the game at any time
- **Responsive Design**: Clean, centered layout that works on different screen sizes

## Technologies Used

- **HTML5**: Structure and content
- **CSS3**: Styling and layout (using CSS Grid)
- **JavaScript (ES6)**: Game logic and interactivity
- **Module Pattern**: Organized code using IIFE modules for GameBoard and Game logic

## How to Play

1. Enter names for Player 1 and Player 2 (optional - defaults to "Player 1" and "Player 2")
2. Click "Start Game" to begin
3. Players take turns clicking on empty squares to place their mark (X or O)
4. The first player to get three marks in a row (horizontally, vertically, or diagonally) wins
5. If all squares are filled without a winner, it's a tie
6. Click "Restart Game" to play again

## Local Development

   ```bash

   ```

2. Navigate to the project directory:
   ```bash
   cd odin-tic-tac-toe
   ```

3. Open `index.html` in your web browser
